HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • (p) is for paragraph or block of text
  • (src) defines the location of the image
  • (img) contains the information about the image
  • (br) breaks up the page
  • (a) creates a link to a webpage
  • (href) points to URL for link

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS 3 ways to style: inline CSS, Internal CSS & External CSS
  • External CSS is the most common used as CSS rules are in seperate files
  • Concept for external CSS known as Seperation of Concerns, seperating code into different files based on use
  • Declerations contain two important components, The CSS Property we want to apply and the Value of property
  • Example for point above, Property=colour and Value=Blue, Property=text align and value=justify
